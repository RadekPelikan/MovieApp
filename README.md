# MovieApp

Admin user can:
- Create new movie
- Create new Room (cinema)
- Create new screening (movie in room)

BFU (běžný Franta uživatel) can:
- See all screenings
- Reserve a seat for a screening
- See all reservations
- Cancel a reservation

### Technologies

- Java 18
- JavaFX (javafx-fxml)
- MySQL
- JDBC
